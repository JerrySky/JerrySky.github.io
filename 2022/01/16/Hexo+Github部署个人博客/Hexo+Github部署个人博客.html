环境准备

  • Git（Mac电脑先安装Xcode）

  • Node.js（需要npm，Node.js自带npm）

  • Hexo

    npm install -g hexo

开始部署

创建并且初始化文件夹目录，Blog可以自定义文件夹名：

hexo init Blog

进入Blog文件夹下，安装相应依赖：

npm install

生成静态文件：

hexo g或者hexo generate

启动本地服务，检查是否成功：

hexo s或者hexo server

访问本地测试地址:

http://localhost:4000

如果看到如下示例界面，说明成功了

同步你的博客到Github

在Github上新建仓库，仓库的名字需要和你的账号对应，格式为: yourname.github.io，其中 「yourname 」就是你的 GitHub 的用户名。

创建一个同步使用的Token：

“Settings”--“Developer settings”--“Personal access tokens“--“Generate new token”

本地进入你的blog的文件夹，编辑_config.yml文件，拉到文件最后：

deploy:
  type: git
  repository: https://github.com/[yourname]/[yourname].github.io.git
  branch: main

安装同步插件：

npm install hexo-deployer-git --save

生成静态文件&同步到Github部署：

hexo g -d

同步过程中会让你输入账号密码，账号就是你的GitHub账号，密码是之前生成的一次性Token，出现如下信息，说明同步成功了。

[info] Deploy done : git

然后浏览器输入[yourname].github.io，如果你看到了和刚才本地访问一样的网页，说明我们的博客已经成功的部署到了 GitHub 了。

绑定个人域名

在/MyGithubBlog/themes/landscape/source目录下创建“CNAME”文件，内容为你自己的域名：

www.xxxxxx.com

最后添加CNAME、A解析
